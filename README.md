This iDescription: YouPrompt
YouPrompt is a modern web application built with Next.js and TypeScript that appears to be a prompt/AI interaction platform. Here are the key details: This is a full-stack TypeScript application designed for an AI-powered prompt platform with payment processing capabilities, user authentication, and a polished modern UI.

Technology Stack
Frontend Framework: Next.js 16.2.3 with React 19.2.4
Language: TypeScript (98.9% of the codebase)
Styling: Tailwind CSS with animation support
UI Components: shadcn/ui with Lucide React icons
State Management & Animations: Framer Motion for smooth interactions
Notifications: React Hot Toast and Sonner for user feedback

Core Features & Integrations
AI Integration: Google Generative AI (@google/generative-ai) for AI-powered functionality
Authentication: NextAuth.js for user authentication and session management
Database: Drizzle ORM with Neon Database (serverless PostgreSQL)
Payment Integration: PhonePe payment gateway support
Security: bcrypt/bcryptjs for password hashing

Development & Deployment
Deployed on Vercel (homepage: https://youprompt.vercel.app)
Includes database seeding scripts for admin setup
Fully configured with ESLint for code quality
Environment-based configuration support
